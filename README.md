# 博客管理中心

> 基于 vue-admin-template 实现网站管理、文章编辑、分类标签管理、文章附件管理等功能，是个人网站的管理端。

**演示Demo:** http://panjiachen.github.io/vue-admin-template


## Build Setup

```bash
# clone the project
git clone https://github.com/githubzlz/personal-website-frontend-admin.git

# enter the project directory
cd personal-website-frontend-admin

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```

## Advanced

```bash
# preview the release environment effect
npm run preview

# preview the release environment effect + static resource analysis
npm run preview -- --report

# code format check
npm run lint

# code format check and auto fix
npm run lint -- --fix
```
